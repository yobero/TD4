Commandes de base git
git add : ajoute un fichier dans le dossier git
git commit : commit les fichier du dossier
git push : mise à jour des fichiers du dossier
Aujourd'hui j'ai appris beaucoup de choses, par exemple :
gitk : fais des graphiques étranges mais amusant
git checkout : fais changer des choses
git show master: voir le fichier local
git show origin/master: voir le fichier dans github
git checkout gh-pages: aller à la branche gh-pages
